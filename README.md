## ⚙️ Tech Stack
- Python
- YOLO (Object Detection)
- CNN (Classification)
- OpenCV
- TensorFlow/Keras
- Flask

---

## 🚀 Features
- Real-time pothole detection  
- Severity classification (Low / Medium / High)  
- Explainable AI with Grad-CAM  
- Batch and single image processing  
- Web-based interface

- Detection Accuracy: **91.5%**  
- Classification Accuracy: **93.2%**
