Frequently Asked Questions
How does the AI pothole detection work? ▾
The system uses a YOLOv8 object detection model to locate potholes in road images, followed by a CNN classifier to grade severity (Moderate / Poor / Severe). GradCAM heatmaps show which regions of the image contributed to the classification decision.
What image formats are supported? ▾
The portal accepts JPG, JPEG, PNG, BMP, and WEBP formats. Maximum file size per image is 50 MB. For batch uploads, select a folder containing road surface images.
How is GPS location determined? ▾
Location is obtained via IP geolocation at the time of upload. For accurate field coordinates, it is recommended that inspection devices have GPS enabled and images contain EXIF metadata with coordinates.
Where are detection results saved? ▾
All results are saved locally to road_data.csv in the application directory, and annotated images are saved to the static/output folder. You can view the full log using the "View Detection Log" button on the Road Monitoring page.
Who can access this portal? ▾
This portal is intended for authorized NHAI field inspectors, highway engineers, and state road transport department officials. Unauthorized access is prohibited under the IT Act, 2000.
